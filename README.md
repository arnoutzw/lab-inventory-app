# Lab Inventory

A comprehensive laboratory inventory management system for tracking equipment, materials, and supplies with powerful search and categorization features.

## Screenshots

[Screenshot placeholder - Inventory dashboard with items and categories]

## Features

- **Inventory Management** - Add, edit, and remove inventory items with detailed information
- **Item Cataloging** - Organize items with descriptions, quantities, and metadata
- **Search Functionality** - Quickly find items using powerful search across all fields
- **Inventory Tracking** - Monitor stock levels and usage over time
- **Category Organization** - Organize items into custom categories for easy navigation

## Tech Stack

- HTML5
- JavaScript
- Progressive Web App (PWA)

## Getting Started

### Quick Start

1. **Online Demo**: Visit [Lab Inventory Live](https://arnoutzw.github.io/lab-inventory-app/)
2. **Local Setup**: Clone the repository and open `index.html` in your web browser

```bash
git clone https://github.com/arnoutzw/lab-inventory-app.git
cd lab-inventory-app
# Open index.html in your browser
```

### Installation

This is a PWA that works entirely in your browser with no server or dependencies required. For best experience:
- Install as a PWA on your device for offline access and quick launch
- Use a modern web browser (Chrome, Firefox, Safari, Edge)

## Usage

1. **Add Items**: Create new inventory entries with details and categories
2. **Search**: Use the search bar to quickly locate items
3. **Manage Stock**: Update quantities and track item locations
4. **Organize**: Group items into categories for better organization
5. **Export**: Generate reports of your inventory
